# Trading Mind Journal

## Why I Wrote This

I wrote this not to preach, but to remember. Every mistake, every emotional hijack, every day I let greed or fear drive
my trades — it taught me something about myself, not the market.

This isn’t about strategies or indicators. It’s about surviving the mental game that quietly kills most traders long
before they blow their capital.

If you’re reading this, take it as a mirror. You’ll see parts of your own journey here.

---

### How to Use

1. Read one file per week — don’t rush.
2. Reflect honestly after each section.
3. Use `04_Journal_Template.md` daily to build emotional consistency.
4. Revisit this journal after every drawdown — it will sound different each time.
