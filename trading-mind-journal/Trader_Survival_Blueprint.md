# Trader Survival Blueprint
_For Ex-Engineers Transitioning to Full-Time Trading_

---

## 🎯 Purpose
This document defines the operating framework for maintaining consistency, discipline, and emotional stability while trading 5-minute ORB and VWAP reclaim setups.

---

## ⚙️ Core Philosophy

- Trading success = *systematic execution × emotional control*  
- Every rule break is a **runtime exception** — handle it immediately.  
- Brutal honesty replaces journaling.  
- Physical punishment replaces guilt (10 pushups + 10 situps × 3 rounds).  
- Consistency matters more than profit during the build phase.

---

## 🧩 Setup Overview

| Component | Description |
|------------|--------------|
| **Chart** | 5-minute timeframe |
| **Strategies** | Opening Range Breakout (primary), VWAP Reclaim (secondary) |
| **Filters** | Sector structure (trending / sideways / opposite) |
| **Indicators** | 9EMA, RSI, structural HH-HL analysis |
| **R Model** | Dynamic targets based on sector context |

---

## 📈 Dynamic R Model

| Sector Context | Definition | Target | Behavior |
|----------------|-------------|---------|-----------|
| 🚫 **Opposite** | Sector making LH–LL, below 9EMA | **2R** | Quick profit, no runners |
| ⚖️ **Sideways** | Flat 9EMA, overlapping candles | **3R** | Partial 2R, trail rest |
| 🚀 **Trending** | HH–HL, strong momentum, 9EMA up | **5R** | Hold till 5R or trail with 9EMA |

**Rule:** Decide R **before entry**, never mid-trade.

---

## 🔁 Daily Loop

### Pre-Market
1. Scan sectors → tag each as trending / sideways / opposite  
2. Set R plan based on context  
3. Recite: `I execute process, not predictions.`

### During Market
- Max 2–3 trades per session  
- For each trade:
  - Check setup validity  
  - Confirm sector context  
  - Execute  
  - Post-trade ask: **“Did I follow my process?”**

If **no** → trigger punishment (pushups/situps).

### Post-Market
- 5-minute debrief:
  - Log: “Followed / Broke process.”  
- No deep journaling. No chart replay until weekend.

---

## 🧠 Weekly Debug Ritual

**Every Saturday (30 mins):**
1. Count rule breaks.  
2. Identify dominant emotion (fear, greed, impatience).  
3. Create one new counter-rule for next week.  
4. **Sunday = chart-free day.**

---

## 📘 Technical Focus

- Restrict trading to **5-min ORBs + VWAP reclaims** only.  
- Do not add indicators for 90 days.  
- Goal: **flawless execution**, not discovery.  
- Treat trading as a codebase: fewer bugs → more stability.
